# Bookstore

Bookstore is a bookstore app, developed using Ruby on Rails.


## Ruby version
```
ruby 2.3.1
rails 5.0
```

## Setting Up
```
git clone https://github.com/netqyq/bookstore
cd bookstore
bundle install
```

## Database initialization
```
rake db:create
rake db:migrate
```

## Run the test suite
```
# start the server
$ rails s
```
